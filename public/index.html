KAJIBOの使い方

家事マニュアルの表示方法

①メニューバー「家事する時」を押す

左から２番目

②メニューバー「個別に見る」を押す

左から3番目

③緑のボタンを押す

例えば「1)異物混入チェック」を押してみる

④家事マニュアルが一つ表示される

再び緑のボタンが出ます

家族を仲間にする方法

①メニューバー「ルーム」を押す

左から3番目

②メニューバー「ルームを作る」を押す

左から2番目

③ルームID長押し後、「転送」を押す

左から4番目

④送信相手を選び、メッセージ入力

メッセージ入力後、右上の転送ボタンを押す

⑤画面右上の三本線ボタンを押す

一番右端のボタン

⑥おすすめを押す

Botをシェアできます

⑦送信相手を選び、メッセージ入力

メッセージ入力後、右上の転送ボタンを押す

仲間になるお誘いを受けたら

①ルームID長押し後、「コピー」を押す

一列目の一番左端

②KAJIBO 家事アシスタントを押す

羊のアイコン押すとBotと友達になれます

③メニューバー「ルーム」を押す

左から3番目

④メニューバー「ルームに入る」を押す

左から3番目

⑤画面左下のキーボードボタンを押す

メニューバーの左端

⑥入力欄に先程のルームIDをペースト

送信するとルームメイトになります

家事マニュアルのカスタマイズ方法

①メニューバー「カスタマイズ」を押す

左から4番目

②メニューバー「登録修正」を押す

左から2番目

③緑のボタンを押す

例えば「4)洗濯ネットの収納場所」を押してみる

④家事マニュアルが一つ表示される

「登録する」を押す

⑤画面左下のキーボードボタンを押す

メニューバーの左端

⑥入力欄に我が家のマニュアルを入力

送信すると登録されます

登録と同時に、ルームメイト全員に登録完了の通知が届きます。

頼まれた家事を終えた時

①メニューバー「家事する時」を押す

左から２番目

②メニューバー「完了報告」を押す

左から4番目

③「はい」ボタンを押す

同じルームに入室している全員に【ご報告】を配信

If you are running the app using Node.js, please set the LIFF ID as an environment variable in your Heroku account follwing the below steps:

  1. Go to `Dashboard` in your Heroku account.
  2. Click on the app you just created.
  3. Click on `Settings` and toggle `Reveal Config Vars`.
  4. Set `MY_LIFF_ID` as the key and the LIFF ID as the value.
  5. Your app should be up and running. Enter the URL of your app in a web browser.

If you are using any other platform, please add your LIFF ID in the index.html file.

For more information about how to add your LIFF ID, see Initializing the LIFF app.
